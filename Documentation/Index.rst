**Backend accessibility analysis modules for TYPO3 13.4 LTS.**

Provides dedicated backend modules that help editors identify and resolve
accessibility issues in editorial content — covering alt texts, ARIA
attributes, heading structure, and link texts — without any external
dependencies.
